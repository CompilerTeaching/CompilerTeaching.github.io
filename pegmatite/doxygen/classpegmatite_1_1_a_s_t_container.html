Detailed Description

type of ast member vector.

The base class for non-leaf AST nodes. Subclasses can have instances of ASTMember subclasses as fields and will automatically construct them.

virtual void pegmatite::ASTContainer::construct ( const InputRange &  r,
ASTStack &  st 
)
virtual

Asks all members to construct themselves from the stack.

The members are asked to construct themselves in reverse order from a node stack (st).

The input range (r) is unused, because the leaf nodes have already constructed themselves at this point.

Implements pegmatite::ASTNode.
